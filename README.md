# Trading-Backtester-Ribbon-Strategy

This  program uses the Binance API TA-Lib  matplotlib and numpy. 

The program is a technical analysis momentum trading strategy.

The strategy uses exponential moving average ribbons for making a buy signal 

and also for making a sell signal. The algorithm was backtested with about 

30% profitable trades and a 200% ROI for a backtest paper trade on a 4h time frame.
